# java-lotto
로또 미션 진행을 위한 저장소

## 기능
- 입력
    - [ ] 구매할 로또 금액 입력
        - [ ] 천 원 미만 절삭
        - [ ] 천 원 미만 시 에러
    - [ ] 지난주 당첨번호 입력
        - [ ] 중복 검사
        - [ ] 범위 검사
    - [ ] 지난주 보너스볼 번호 입력
        - [ ] 중복 검사
        - [ ] 범위 검사
    
- 출력
    - [ ] 구매한 로또번호 출력기능
    - [ ] 요구사항 출력기능
        - [ ] 금액입력 요구
        - [ ] 당첨번호 입력 요구
        - [ ] 보너스볼 입력 요구
    - [ ] 당첨통계 출력기능
    
- 로또 계산기 (enum)
    - [ ] 수익률 계산 기능
    - [ ] 당첨번호 비교 기능
    
- 랜덤 유틸리티
    - [ ] 번호 생성기
        - [ ] 6개의 수 생성
        - [ ] 1~45 사이 수 중복 제외
        - [ ] 정렬 기능
